
# Netflix Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Goals
- Analyze distribution of Movies vs TV Shows
- Explore release trends across years
- Identify top countries producing Netflix content
- Understand popular genres and ratings

## Key Insights
- Netflix content increased rapidly after 2015
- The United States produces the highest number of titles
- Drama and International Movies are among the most common genres
- Movies dominate the platform compared to TV Shows

## Files
- `netflix_analysis.ipynb` – main analysis notebook
- `netflix_titles.csv` – dataset
